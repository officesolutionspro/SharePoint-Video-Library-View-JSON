# SharePoint-Video-Library-View-JSON
SharePoint Video Library JSON

**Required Fields**
Title (Existing) 
Name (Existing) 
Created by (Existing) 
Video Description (multiple lines) 
Date (Date Only) 
Like (Person show pictures—no groups—allow multiple selections) 
Disklike (Person show pictures—no groups—allow multiple selections) 
Thumbnail (Image) 

Notes: Created by must be added to the view for the Edit Details to be visible.  I recommend all fields be added to the view before adjusting the JSON.  

Enjoy!
